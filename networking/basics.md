# 🌐 Networking Basics

## ⚔️ TCP vs UDP

### TCP
- Reliable
- Ordered delivery
- Used for APIs, RPC, web

### UDP
- Fast
- No guarantees
- Used for lightweight communication

---

## 🔌 Key Ports

- 8545 → JSON-RPC (query node)
- 8551 → Engine API (consensus client)
- 30303 → P2P network (peer connections)

---

## 📡 RPC

- RPC = Remote Procedure Call
- Allows you to request data from a node

---

## 📦 JSON-RPC

- Uses JSON format for RPC requests
- Example: eth_blockNumber
