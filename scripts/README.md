# Operator Scripts

These scripts help me operate and monitor my Ethereum node stack:

- `service-status.sh` → checks systemd service status
- `disk-memory-check.sh` → checks storage and RAM
- `quick-log-summary.sh` → shows recent logs
- `health-check.sh` → checks services, ports, disk, memory, Prometheus, and Grafana
- `restart-verify.sh` → restarts services and verifies recovery
- `daily-operator-check.sh` → runs the main daily routine
